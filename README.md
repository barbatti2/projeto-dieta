Projeto Dieta - Aplicativo Web Completo

Crie um aplicativo web completo chamado “Projeto Dieta”.

Objetivo:
Ajudar usuários a controlar alimentação, calorias, peso, exercícios e evolução física de forma simples, bonita e intuitiva.

Design

* Visual moderno e profissional.
* Cores predominantes em azul.
* Tema fitness, saúde, academia e resultados.
* Layout responsivo para celular, tablet e computador.
* Interface semelhante aos melhores aplicativos de dieta e treino.
* Animações suaves.
* Bordas arredondadas.
* Aparência premium.

Logo

Utilizar logotipo com identidade fitness e a frase:

“Foco • Treino • Resultados”

Tela Inicial

Ao entrar no sistema, exibir:

* Resumo das calorias do dia.
* Peso atual.
* Meta de peso.
* Consumo de água.
* Exercícios realizados.
* Calendário de acompanhamento.
* Gráfico de evolução.

Sistema de Login

Criar:

* Login por e-mail e senha.
* Cadastro de usuário.
* Recuperação de senha.
* Opção “Lembrar-me”.
* Manter usuário conectado mesmo após fechar o navegador.

Perfil do Usuário

Permitir cadastro de:

* Nome
* Sexo
* Idade
* Altura
* Peso atual
* Peso objetivo
* Nível de atividade física

O sistema deve calcular automaticamente:

* TMB
* Gasto calórico diário
* Meta de calorias para emagrecimento

Diário Alimentar

Separar refeições em:

* Café da Manhã
* Almoço
* Pré-Treino
* Jantar
* Ceia

Cada refeição deve permitir:

* Buscar alimento
* Informar quantidade em gramas
* Adicionar observações

Mostrar:

* Calorias
* Proteínas
* Carboidratos
* Gorduras

Atualizar os totais automaticamente.

Banco de Alimentos

Criar banco com centenas de alimentos brasileiros.

Incluir pelo menos:

* Ovo
* Frango
* Arroz
* Feijão
* Cenoura
* Pão Integral
* Leite
* Banana
* Maçã
* Batata Doce
* Carne Bovina
* Peixes
* Queijos
* Iogurtes

Permitir busca rápida.

Histórico

Criar histórico completo.

O usuário deve poder:

* Ver qualquer dia anterior.
* Editar registros.
* Excluir registros.
* Filtrar por período.
* Exportar dados.

Calendário

Criar calendário visual estilo FatSecret.

Utilizar cores:

Verde:

* Dieta cumprida.
* Treino realizado.

Azul:

* Apenas treino realizado.

Laranja:

* Apenas dieta cumprida.

Vermelho:

* Dieta não cumprida.

Ao clicar em qualquer dia, abrir os detalhes.

Controle de Peso

Criar área para:

* Registro de peso.
* Medidas corporais.
* Fotos de evolução.

Exibir:

* Gráfico semanal.
* Gráfico mensal.
* Gráfico anual.

Controle de Água

Permitir registrar consumo diário.

Meta configurável.

Exibir progresso visual.

Treinos

Permitir registrar:

* Musculação.
* Cardio.
* Caminhada.
* Corrida.
* Bicicleta.

Mostrar histórico e estatísticas.

Dashboard

Criar dashboard com:

* Calorias consumidas.
* Calorias restantes.
* Peso perdido.
* Dias consecutivos de dieta.
* Dias consecutivos de treino.
* Evolução semanal.
* Evolução mensal.

Notificações

Permitir lembretes:

* Beber água.
* Fazer refeição.
* Registrar peso.
* Treinar.

Instalação como Aplicativo

Transformar o sistema em PWA.

Requisitos:

* Instalação no Android.
* Instalação no iPhone.
* Funcionamento semelhante a aplicativo.
* Ícone personalizado.
* Splash screen personalizada.

Banco de Dados

Utilizar banco de dados em nuvem.

Todos os dados devem permanecer salvos após logout.

Administração

Criar painel administrativo com:

* Quantidade de usuários.
* Usuários ativos.
* Relatórios.
* Estatísticas gerais.

Performance

* Carregamento rápido.
* Código otimizado.
* SEO configurado.
* Compatível com Chrome, Safari, Edge e Firefox.

Tecnologias

Utilizar:

* React
* Next.js
* TypeScript
* Tailwind CSS
* Supabase
